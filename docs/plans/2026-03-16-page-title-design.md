# ページタイトル変更 設計

**目的**
- Streamlit ページ上に表示されるアプリタイトルを `一級建築士 クイズアプリ` から `法規クイズアプリ` に変更する。

**対象**
- `src/quiz_app/app.py` の `st.title(...)`
- `tests/test_app.py` の初期表示テスト

**非対象**
- ブラウザタブ名
- README やその他ドキュメント文言
- クイズデータ、採点ロジック、画面遷移

**設計方針**
- 変更は表示タイトル文字列のみに限定する。
- 既存の UI 構造は変えない。
- 回帰防止のため、初期画面タイトルの既存テストを新文言へ更新する。

**確認方法**
- `tests/test_app.py` の該当テストでタイトル文字列を検証する。
- 仕上げに `ruff check`、`ruff format --check`、`pytest` を実行する。
